## 📌 Project Title

**To-Do List App with Local Storage**

## 🎯 Objective

The objective of this project is to create a simple task management web application that allows users to add, delete, and mark tasks as completed. The application uses **browser Local Storage** to ensure that tasks remain saved even after refreshing or closing the browser.

---

## 📖 Project Description

This To-Do List application helps users manage their daily activities efficiently. It provides a simple interface where users can add tasks, mark them as completed, and delete them when finished.

The application stores tasks in the browser using **Local Storage**, which means the data persists even after refreshing the page.

This concept is widely used in **productivity tools and task management applications**.

---

## 🧠 How It Works

1. Users enter a task in the input field.
2. Clicking **Add** creates a new task in the list.
3. Clicking a task marks it as **completed**.
4. Clicking **X** deletes the task.
5. All tasks are stored in **Local Storage**, so they remain saved after refreshing the page.
